geckoDM

A simple Google Chrome Extension designed to help you download multiple lectures at once.
geckoDM targets the 2017 release of Echo360, for older versions this works.

  •   Google Chrome
  •   FireFox (Coming Soon!)

Download and Install Extension

Step 1: Install our Extension over the Chrome Web Store.

  •   Install Extension
  •   Install Extension (Coming Soon!)

Step 2: Go to the Echo360 Page Provided by Your Instituion

Step 3: Load the Echo page (via the link), click on the Gecko extension

  1. Click on load lectures
  2. Select Lectures to Download
  3. Click on the Download button

Step 4: Wait for Download to Finish

Step 5: Watch the Lecture. Its that simple!
